Monster Island

Monster Island is a game that I made by my self in the 9th grade with the aim of attempting to release it on steam.
Unfortunatly it was never released, but a playable demo of the game was completed.

Overview

Monster Island is a pc game that I created with the usage of C# programming language, Javascript programming language, and the Unity 3d game engine. It is a horror first person survival game set in an island surrounded by mountains, and within it a haunted house that was completed as well as a forest with different types of trees. A floating enemy monster chases and attacks the player as the player attempts to kill it with their axe. The player can lose if the enemy manages to touch them or win if they are able to kill the enemy before the enemy can touch them.

MonsterIsland 0.01 Video

A series of videos were created in order to keep track of the progress of Monster Island. The 0.01 Version of Monster Island showed off the newly created forest as well as the flashlight, and the skybox. This early stage of the game was focused at creating the scene for the game.

How Monster Island was Created

I created Monster Island myself, using the Unity 3d Engine, I scripted it using Javascript, and C#, and I used blender to design some of the map. I also made use of the Unity store library in order to add props such as weapons and other items.

This video displays the first haunted house added to the game. The haunted house was entirely created by me with Unity 3d and a little bit of Blender.
