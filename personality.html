Personality Test Application

The Personality Test application is an an app that I created in my Senior Year of high school for the AP Computer Science Principles course.

Overview

  • The app was created entirely in Javascript with the usage of code.org's code studio.
  • This is a Personality Test app. The idea is that you answer some questions and then I created a system that adds and subtracts points based on the choice of the user and then based on that your personality type is revealed.
  • There are 4 types of Personalities, and the explanation of the specific personality type of the user is revealed to the user once they have completed answering the questions.
  • The application received a score of 4/5 by the College Board. For further understanding of this score, please conteact me.
